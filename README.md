Real-time banner system with admin dashboard, live updates via WebSockets, and countdown-based auto-hide.

---

## 🚀 Features
- Admin dashboard to update banner content  
- Real-time updates using Socket.IO  
- Countdown timer with auto-hide  
- Visibility toggle for banner  
- MySQL database integration  

---

## 🛠️ Tech Stack
- Frontend: React  
- Backend: Node.js, Express  
- WebSockets: Socket.IO  
- Database: MySQL  


# Create a .env file in /server:

DBHOST=your_host
DBPORT=your_port
DBUSER=your_user
DBPASSWORD=your_password
DBDATABASE=your_database
PORT=9000
